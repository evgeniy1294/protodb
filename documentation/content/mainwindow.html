Окно программы

Окно приложения можно условно поделить на три части: боковая панель с гамбургер-меню(1), окно сессии (2) и таблицы последовательностей(3). Приложение построено с использованием библиотеки Qt ADS, что позволяет пересобирать раскладку окна индивидуально под пользователя. При перезапуске программы раскладка сохраняется.

При необходимости, окно сессии или любую из таблиц можно закрыть. Восстановить закрытые окна можно через гамбургер-меню, раздел widgets.
